第一章 疯狂年代

中国，1967年。

红色联合的武斗中，叶文洁目睹了父亲叶哲泰被批斗致死的场景。那一天，整个世界都在疯狂之中。革命的烈火烧遍大地，理智和文明在口号声中灰飞烟灭。

叶文洁站在人群中，看着父亲被推上批斗台。她的眼泪早已流干，只剩下一种冰冷的麻木。她知道，从这一刻起，她曾经相信的一切都崩塌了。

多年后，当她站在红岸基地的发射天线前，仰望着那面巨大的抛物面天线指向深邃的星空时，她做出了一个改变人类命运的决定。

她按下了发射键，将那个来自地球的信号，以光速向宇宙深处发送出去。她不知道，这个信号将在四光年外的三体世界掀起怎样的波澜。
