第一节

我比现在年轻十岁的时候，获得了一个游手好闲的职业，去乡间收集民间歌谣。那一年的整个夏天，我如同一只乱飞的麻雀，游荡在知了和阳光充斥的田野之中。

我就是在那时候遇到了那个名叫福贵的老人。他正赶着一头老牛在犁田，嘴里吆喝着："做牛耕田，做狗看家，做和尚化缘，做鸡报晓，做女人织布。"

那头牛叫福贵，和它的主人同名。老人黝黑的脸上布满了皱纹，像是被岁月犁过无数遍的土地。他的眼睛浑浊，但偶尔闪过的光芒，却像是藏着整个世界的悲欢离合。

他坐在田埂上，抽着旱烟，开始讲述他的一生。从富家少爷到一贫如洗的农民，从妻儿环绕到孤身一人，他用平静的语调讲述着那些足以让人心碎的故事。
